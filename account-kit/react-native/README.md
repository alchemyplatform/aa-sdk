# `@account-kit/react-native`

> **Note:** We recommend using [`@alchemy/wallet-apis`](https://www.npmjs.com/package/@alchemy/wallet-apis) instead, which is currently in beta but recommended for new integrations.

React Native components and hooks for Alchemy Account Kit. Provides pre-built UI components and React Native hooks for integrating smart accounts into mobile applications.

## Getting started

```bash
npm i @account-kit/react-native
```
