# @account-kit/react-native-signer

React Native compatible Alchemy signer impl

## Installation

```sh
npm install @account-kit/react-native-signer
```

## Usage

```js
import { multiply } from "@account-kit/react-native-signer";

// ...

const result = multiply(3, 7);
```
