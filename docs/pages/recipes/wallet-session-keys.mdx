## Quick Start

Set up and run this demo on your machine:

1. **Clone and setup:**

   <CodeBlocks>

   ```bash HTTPS
   git clone https://github.com/alchemyplatform/wallet-session-key-app
   cd wallet-session-key-app
   pnpm install
   ```

   ```bash SSH
   git clone git@github.com:alchemyplatform/wallet-session-key-app.git
   cd wallet-session-key-app
   pnpm install
   ```

   ```bash "GitHub CLI"
   gh repo clone alchemyplatform/wallet-session-key-app
   cd wallet-session-key-app
   pnpm install
   ```

   </CodeBlocks>

2. **Configure environment:**

   ```bash
   cp .env.example .env
   # Fill in your ALCHEMY_API_KEY and PAYMASTER_POLICY_ID
   ```

3. **Initialize application**

   ```bash
   pnpm run dev
   ```

## Success!

You can now interact with Aave protocol through smart wallets with sponsored gas fees, enabling seamless DeFi experiences for supplying, withdrawing, and managing liquidity without gas complexity.
