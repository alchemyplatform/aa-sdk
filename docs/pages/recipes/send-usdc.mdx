In this recipe you'll construct an ERC-20 `transfer` call and submit it through a Smart Account Client. The same pattern works for **any** ERC-20 token; just swap the token address and number of decimals.

> Prefer code? Jump straight to the **React** or **Core** tabs below.

## Prerequisites

1. Account Kit integrated in your app (see the [React quickstart](/wallets/react/quickstart) or [Core quickstart](/wallets/core/quickstart)).
2. A connected user with an on-chain smart account.
3. The USDC contract address for your target chain. For Ethereum mainnet this is `0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48`.

## 1. Encode the `transfer` calldata

```ts
import { encodeFunctionData, parseAbi } from "viem";

const usdcAddress = "0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48";
const recipient = "0xRecipientAddress";
const amount = 10n * 1_000_000n; // 10 USDC (6 decimals)

const transferCalldata = encodeFunctionData({
  abi: parseAbi(["function transfer(address,uint256) returns (bool)"]),
  functionName: "transfer",
  args: [recipient, amount],
});
```

## 2. Send the transaction

<Tabs>
  <TabItem value="react" label="React (hook)">

```tsx
import { useSmartAccountClient } from "@alchemy/aa-react";

export function SendUsdcButton() {
  const { data: client } = useSmartAccountClient();

  const handleClick = async () => {
    if (!client) return;

    const txHash = await client.sendTransaction({
      to: usdcAddress,
      data: transferCalldata,
      value: 0n, // no native value for ERC-20 transfers
    });

    console.log("USDC transfer userOp hash:", txHash);
  };

  return <button onClick={handleClick}>Send USDC</button>;
}
```

## 3. Wait for the transaction to be mined

If you're using the React hooks, you can track confirmation with `useWaitForUserOperationTransaction`:

```ts
import { useWaitForUserOperationTransaction } from "@alchemy/aa-react";

const { waitForUserOp } = useWaitForUserOperationTransaction();
await waitForUserOp(txHash);
```
