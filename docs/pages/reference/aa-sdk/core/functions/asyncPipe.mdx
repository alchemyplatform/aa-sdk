Utility function that allows for piping a series of async functions together

## Import

```ts
import { asyncPipe } from "@aa-sdk/core";
```

## Parameters

### fns

`((s: S, o?: O, f?: F) => Promise<S>)[]`

- functions to pipe

## Returns

`S`
result of the pipe
