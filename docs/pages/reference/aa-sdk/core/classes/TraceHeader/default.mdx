Creating a default trace id that is a random setup for both trace id and parent id

## Import

```ts
import { TraceHeader } from "@aa-sdk/core";
```

## Usage

```ts
const traceHeader =
  TraceHeader.fromTraceHeader(headers) || TraceHeader.default();
```

## Returns

`TraceHeader`
A default trace header
