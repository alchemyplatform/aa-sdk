Creates an instance of an error with a message indicating an invalid RPC URL.

  supported methods.
</Note>

## Import

```ts
import { InvalidRpcUrlError } from "@aa-sdk/core";
```

## Parameters

### rpcUrl

`string`
The invalid RPC URL that caused the error
