Throws an error when the client type does not match the expected client type.

<Note>
  `IncompatibleClientError` extends `BaseError`, see the docs for BaseError for
  all supported methods.
</Note>

## Import

```ts
import { IncompatibleClientError } from "@aa-sdk/core";
```

## Parameters

### expectedClient

`string`
The expected type of the client.

### method

`string`
The method that was called.

### client

`Client`
The client instance.
