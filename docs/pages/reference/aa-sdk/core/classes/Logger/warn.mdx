Logs a warning message if the logging conditions are met.

## Import

```ts
import { Logger } from "@aa-sdk/core";
```

## Usage

```ts
import { Logger } from "@aa-sdk/core";

Logger.warn("Careful...");
```

## Parameters

### msg

`string`
The message to log as a warning

### args

`...any[]`
Additional parameters to log along with the message
