Sets the log filter pattern.

## Import

```ts
import { Logger } from "@aa-sdk/core";
```

## Usage

```ts
import { Logger } from "@aa-sdk/core";

Logger.setLogFilter("error");
```

## Parameters

### pattern

`string`
The pattern to set as the log filter
