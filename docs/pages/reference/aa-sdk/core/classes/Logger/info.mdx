Logs an informational message to the console if the logging level is set to INFO.

## Import

```ts
import { Logger } from "@aa-sdk/core";
```

## Usage

```ts
import { Logger } from "@aa-sdk/core";

Logger.info("Something is happening");
```

## Parameters

### msg

`string`
the message to log

### args

`...any[]`
additional arguments to log alongside the message
