Constructs an error message indicating that no default factory was found for the given account type, chain, and entry point version.

<Note>
  `DefaultFactoryNotDefinedError` extends `BaseError`, see the docs for
  BaseError for all supported methods.
</Note>

## Import

```ts
import { DefaultFactoryNotDefinedError } from "@aa-sdk/core";
```

## Parameters

### accountType

`string`
the type of account

### chain

`Chain`
the blockchain chain

### version

`EntryPointVersion`
the entry point version
