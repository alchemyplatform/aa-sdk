Constructs an error message indicating that batch execution is not supported by the specified account type.

<Note>
  `BatchExecutionNotSupportedError` extends `BaseError`, see the docs for
  BaseError for all supported methods.
</Note>

## Import

```ts
import { BatchExecutionNotSupportedError } from "@aa-sdk/core";
```

## Parameters

### accountType

`string`
the type of account that does not support batch execution
