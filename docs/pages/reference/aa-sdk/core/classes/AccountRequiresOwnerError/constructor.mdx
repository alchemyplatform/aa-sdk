Constructs an error indicating that an account of the specified type requires an owner to execute.

<Note>
  `AccountRequiresOwnerError` extends `BaseError`, see the docs for BaseError
  for all supported methods.
</Note>

## Import

```ts
import { AccountRequiresOwnerError } from "@aa-sdk/core";
```

## Parameters

### accountType

`string`
The type of account that requires an owner
