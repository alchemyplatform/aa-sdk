```ts
type Prettify<T> = { [K in keyof T]: T[K] } & object;
```

Defined in: [aa-sdk/core/src/utils/types.ts:41](https://github.com/alchemyplatform/aa-sdk/blob/5b9429f0f16dcd4ff822527b8de589e19e88faa7/aa-sdk/core/src/utils/types.ts#L41)

## Example

```ts
Prettify<{ a: string } & { b: string } & { c: number, d: bigint }>
=> { a: string, b: string, c: number, d: bigint }
```
