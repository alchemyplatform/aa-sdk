Adapts a Wallet Standard wallet into a [SolanaSigner](../../interfaces/SolanaSigner).

Works with any wallet that implements the `solana:signTransaction` feature
from the Wallet Standard spec (Phantom, Solflare, etc.). No peer dependency
required — the interface uses `Uint8Array` directly.

For `@solana/wallet-adapter-react` wallets, use [fromWalletAdapter](fromWalletAdapter).
For `@solana/kit` signers, use [fromKitSigner](fromKitSigner). For raw Ed25519
keypairs, use [fromKeypair](fromKeypair).
