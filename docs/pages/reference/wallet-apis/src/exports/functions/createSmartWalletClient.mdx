Defined in: [packages/wallet-apis/src/client.ts:116](https://github.com/alchemyplatform/aa-sdk/blob/v5.x.x/packages/wallet-apis/src/client.ts#L116)

Creates a smart wallet client for EVM or Solana chains.

Pass an EVM `Chain` to get a [SmartWalletClient](../type-aliases/SmartWalletClient) with EVM actions.
Pass a Solana chain ID string (e.g. `"solana:devnet"`) with a
[SolanaSigner](../interfaces/SolanaSigner) to get a [SolanaSmartWalletClient](../type-aliases/SolanaSmartWalletClient).

Imported from `@alchemy/wallet-apis` — works for both chains.
Solana-specific actions and adapters are in `@alchemy/wallet-apis/solana`.
