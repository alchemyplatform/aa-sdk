Creates an API key that can take any action on behalf of the current user.
(Note that this method is currently experimental and is subject to change.)

## Import

```ts
import { BaseSignerClient } from "@account-kit/signer";
```

## Parameters

### params

`CreateApiKeyParams`
Parameters for creating the API key.

### params.name

`string`
Name of the API key.

### params.publicKey

`string`
Public key to be used for the API key.

### params.expirationSec

`number`
Number of seconds until the API key expires.
