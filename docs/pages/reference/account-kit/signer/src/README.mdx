| [AlchemyServerSigner](classes/AlchemyServerSigner.mdx)       | AlchemyServerSigner is a signer that can sign messages and typed data using an access key. It extends the SmartAccountSigner interface and uses the ServerSignerClient to sign requests. Primarily intended to be used server-side. |
| [AlchemySignerWebClient](classes/AlchemySignerWebClient.mdx) | A lower level client used by the AlchemySigner used to communicate with Alchemy's signer service.                                                                                                                                   |

| [OAuthProvidersError](classes/OAuthProvidersError.mdx)       | -                                                                                                                                                                                                                                   |
