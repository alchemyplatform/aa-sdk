Defined in: [account-kit/core/src/actions/watchChain.ts:19](https://github.com/alchemyplatform/aa-sdk/blob/5b9429f0f16dcd4ff822527b8de589e19e88faa7/account-kit/core/src/actions/watchChain.ts#L19)

Allows you to subscribe to changes of the chain in the client store.

## Example

```ts
import { watchChain } from "@account-kit/core";
// see createConfig for more information on how to create a config
import { config } from "./config";

watchChain(config)(console.log);
```
