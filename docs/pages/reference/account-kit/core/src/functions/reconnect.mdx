```ts
function reconnect(config): Promise<void>;
```

Defined in: [account-kit/core/src/actions/reconnect.ts:19](https://github.com/alchemyplatform/aa-sdk/blob/49e74eee8606f480b200b4612d00d950da8a8742/account-kit/core/src/actions/reconnect.ts#L19)

This method will use the current state in the client store and attempt to restore
connected instances of previously used accounts and the signer.

## Example

```ts
import { reconnect } from "@account-kit/core";
import { config } from "./config";

await reconnect(config);
```

## Parameters

| Parameter | Type                                                                 | Description                                        |
| :-------- | :------------------------------------------------------------------- | :------------------------------------------------- |
| `config`  | [`AlchemyAccountsConfig`](../type-aliases/AlchemyAccountsConfig.mdx) | the account config which contains the client store |

## Returns

`Promise`\<`void`>
