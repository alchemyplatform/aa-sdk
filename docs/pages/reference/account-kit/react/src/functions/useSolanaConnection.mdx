function useSolanaConnection(opts): SolanaConnection;

Defined in: [account-kit/react/src/hooks/useSolanaConnection.ts:39](https://github.com/alchemyplatform/aa-sdk/blob/923a52171f7246172caf350853843e6680197da9/account-kit/react/src/hooks/useSolanaConnection.ts#L39)

This hook is used for establishing a connection to Solana and returns the connection object and the signer object.

## Example

```ts twoslash
import { useSolanaConnection } from "@account-kit/react";

const { connection } = useSolanaConnection();
```

## Parameters

| Parameter | Type                         | Description                       |
| :-------- | :--------------------------- | :-------------------------------- |
| `opts`    | `SolanaConnectionHookParams` | Options for the hook to get setup |

## Returns

[`SolanaConnection`](../interfaces/SolanaConnection.mdx)

The transaction hook.
