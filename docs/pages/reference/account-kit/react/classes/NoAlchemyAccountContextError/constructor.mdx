Constructs an error message indicating that a specific hook must be used within an AlchemyAccountProvider.

<Note>
  `NoAlchemyAccountContextError` extends `BaseError`, see the docs for BaseError
  for all supported methods.
</Note>

## Import

```ts
import { NoAlchemyAccountContextError } from "@account-kit/react";
```

## Parameters

### hookName

`string`
The name of the hook that must be used within the AlchemyAccountProvider
