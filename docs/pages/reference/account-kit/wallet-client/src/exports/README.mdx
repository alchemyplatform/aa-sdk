If an account already exists for the signer, it will always return that account unless a new ID is specified. If an account already exists, the creationHint will be ignored. |

This method requests the account associated with the signer and uses it to sign the message.                                                                                                                  |

This method handles different types of signature requests including personal_sign, eth_signTypedData_v4, and authorization.                                                                       |
| [signTypedData](functions/signTypedData.mdx)                     | Signs typed data (EIP-712) using the smart account. This method requests the account associated with the signer and uses it to sign the typed data.                                                                                                    |
| [waitForCallsStatus](functions/waitForCallsStatus.mdx)           | Waits for the status of a prepared call to be updated, returning after the calls are no longer pending.                                                                                                                                                |
