Waits for the status of a prepared call to be updated, returning after the calls are no longer pending.

## Example

```ts
const result = await client.waitForCallsStatus({ id: "0x1234..." });
console.log(result);
```

## Parameters

| Parameter | Type                           | Description                                                                             |

| `client`  | `InnerWalletApiClient`         | The wallet API client to use for the request                                            |
| `params`  | `WaitForCallsStatusParameters` | Parameters for waiting for calls status, including the call ID and options for polling. |
