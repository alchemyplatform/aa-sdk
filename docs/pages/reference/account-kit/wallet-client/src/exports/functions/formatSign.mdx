Defined in: [account-kit/wallet-client/src/client/actions/formatSign.ts:41](https://github.com/alchemyplatform/aa-sdk/blob/da19afd63b0ba06a4de95b0d2d7871f64d546304/account-kit/wallet-client/src/client/actions/formatSign.ts#L41)

Formats a signature request for signing messages or transactions.

## Example

```ts
// Formats a signature
const result = await client.formatSign({
  from: "0x1234...",
  signature: {
    type: "ecdsa",
    data: "0xabcd...",
  },
});
```
