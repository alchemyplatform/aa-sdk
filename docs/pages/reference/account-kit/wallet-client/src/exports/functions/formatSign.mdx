Defined in: [account-kit/wallet-client/src/client/actions/formatSign.ts:41](https://github.com/alchemyplatform/aa-sdk/blob/49e74eee8606f480b200b4612d00d950da8a8742/account-kit/wallet-client/src/client/actions/formatSign.ts#L41)

Formats a signature request for signing messages or transactions.

## Example

```ts
// Formats a signature
const result = await client.formatSign({
  from: "0x1234...",
  signature: {
    type: "ecdsa",
    data: "0xabcd...",
  },
});
```
