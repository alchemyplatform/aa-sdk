## Prerequisites

- `React 18` or higher
- `TypeScript 5` or higher

## Installation

Install the [`@account-kit/infra`](https://www.npmjs.com/package/@account-kit/infra) and [`@account-kit/react`](https://www.npmjs.com/package/@account-kit/react) packages from Smart Wallets along with required dependencies:

<CodeBlocks>

```bash npm
npm install @account-kit/infra @account-kit/react @tanstack/react-query
npm install -D tailwindcss @tailwindcss/postcss postcss
```

```bash pnpm
pnpm add @account-kit/infra @account-kit/react @tanstack/react-query
pnpm add -D tailwindcss @tailwindcss/postcss postcss
```

```bash yarn
yarn add @account-kit/infra @account-kit/react @tanstack/react-query
yarn add -D tailwindcss @tailwindcss/postcss postcss
```

</CodeBlocks>
