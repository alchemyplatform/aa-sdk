# useUser

The `useUser` hook returns the authenticated [`User`](/resources/types#user) if the signer is authenticated.

## Import

```ts
import { useUser } from "@account-kit/react";
```

## Usage

```tsx [useUser.tsx]
// [!include ~/snippets/react/useUser.tsx]
```

## Return Type

```ts
import { type UseUserResult } from "@account-kit/react";
```

Returns a `User` object if the user has been authenticated, othwerise `null`.
