# useBundlerClient

The `useBundlerClient` hook returns the underlying Bundler RPC client instance.

## Import

```ts
import { useBundlerClient } from "@account-kit/react";
```

## Usage

```tsx [useBundlerClient.tsx]
// [!include ~/snippets/react/useBundlerClient.tsx]
```

## Return Type

```ts
import { type UseBundlerClientResult } from "@account-kit/react";
```

Returns an instance of `ClientWithAlchemyMethods` which is the JSON RPC client connected to Alchemy services.
